# UI Style Guide (Temel)

## 1) Renk Paleti (Pastel, 4–6 ana renk)
- **Primary (Mint):** #6EC3B2
- **Secondary (Ice Blue):** #A5D8FF
- **Accent (Blush Pink):** #FFC9DE
- **Info (Lavender):** #C7CFFF
- **Success (Soft Green):** #8CD790
- **Warning (Warm Peach):** #FFD8A8

**Nötrler**
- Background:** #F8FAFC
- Surface (Kart):** #FFFFFF
- Text Primary:** #1F2937
- Text Muted:** #6B7280
- Divider/Border:** #E5E7EB

## 2) Tipografi
- **Başlık:** *Playfair Display*
  - H1: 32px / semibold
  - H2: 24px / semibold
- **Metin:** *Inter*
  - Body: 16px / regular
  - Small: 14px / regular
- **Satır yüksekliği:** Başlık 120%, Metin 150%
- **Letter-spacing:** Normal; H1/H2’de -0.2px

## 3) Buton Stilleri
- **Yapı:** border-radius: 12px; padding: 12px 16px; font: Inter 16px semibold
- **Renkler:** 
  - Primary: bg **#6EC3B2**, text **#0F172A**
  - Hover: bg **#5AB6A4** (≈ %8 koyulaştır)
  - Disabled: bg **#E5E7EB**, text **#9CA3AF**
- **Border:** 1px solid **#D1D5DB** (opsiyonel)
- **Gölge:** 0 2px 6px rgba(0,0,0,0.06)
- **Focus:** 2px **#A5D8FF** dış parlama (accessible)

## 4) Kart (Card) Stili
- **Arka plan:** **#FFFFFF**
- **Köşeler:** 16px
- **İç boşluk:** 16–20px
- **Gölge:** 0 1px 4px rgba(0,0,0,0.05) (hover: 0 4px 12px rgba(0,0,0,0.08))
- **Ayrım (divider):** 1px **#E5E7EB**
- **Başlık:** Playfair 18–20px semibold; Metin: Inter 14–16px
- **Durum renkleri:** success **#8CD790**, warning **#FFD8A8**, info **#C7CFFF**

## 5) Genel İlkeler
- Soft, okunabilir, minimalist.
- Kontrast oranı en az 4.5:1 (metinler için).
- Boşluk sistemleri: 4/8’lik grid (4, 8, 12, 16, 20, 24…).

## 6) İkon ve İlave Notlar
- İkonlar: tek renk **#334155** veya metin rengine uyumlu.
- Yuvarlak köşeler ve yumuşak gölgeler her yerde tutarlı.
